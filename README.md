# 工会固定资产管理系统 V18.0 用户操作手册

* * *

## 一、系统简介

**工会固定资产管理系统 V18** 是一款符合财政部《工会会计制度》（财会〔2021〕7号）的专业资产管理软件，支持固定资产的全生命周期管理，包括资产登记、折旧计算、查询统计、Excel导入导出等功能。

### 主要特性

* ✅ **符合工会会计制度**：折旧计算严格遵循财政部规定
* ✅ **智能编码生成**：按 GB/T 14885-2022 自动生成国标分类编码
* ✅ **精准折旧计算**：月折旧额四舍五入，差额首月一次性处理
* ✅ **Excel 批量导入**：支持从模板批量导入资产数据
* ✅ **多维度查询**：支持按编号、名称、类别、部门等条件查询
* ✅ **月度折旧表**：自动生成月度折旧明细和汇总报表
* ✅ **数据导出**：支持导出资产清单、折旧报表到 Excel

## 二、系统安装

### 2.1 环境要求

| 组件  | 要求  | 说明  |
| --- | --- | --- |

### 2.2 Python 安装

1. 访问官网下载 Python：https://www.python.org/downloads/

3. 点击 "Install Now" 完成安装

### 2.3 安装 openpyxl（可选，用于 Excel 功能）

打开命令提示符（CMD），执行：

    pip install openpyxl

### 2.4 软件安装

1. 解压 `工会固定资产管理系统_v17.0_最终稳定版.zip`
  
2. 将解压后的文件夹放到任意位置（建议：D:\工会资产管理系统）
  
3. 文件夹结构：
  
      工会资产管理系统/
      ├── main.py          # 主程序
      ├── run.bat          # 启动脚本
      └── V18.0_版本说明.txt

## 六、资产管理

### 6.1 新增资产

1. 点击工具栏"➕ 新增"按钮
2. 填写资产信息：
  * **资产编号**：留空，系统自动生成
  * **资产名称**：必填，如"联想笔记本电脑"
  * **资产类别**：必填，从下拉列表选择
  * **原值**：必填，如"5000"
  * **购置日期**：必填，格式 YYYY-MM-DD
  * **使用部门**：选填
  * **使用人**：选填
  * **存放地点**：选填
  * **状态**：默认"在用"
  * **备注**：选填
3. 点击"保存"按钮

**系统自动生成编码规则**：

    国标分类码-YY-MM-DD-序号
    示例：A02010101-26-03-27-001

### 6.2 编辑资产

1. 在资产列表中选中要修改的资产
2. 点击工具栏"✏️ 编辑"按钮
3. 修改相关信息
4. 点击"保存"按钮

**注意**：资产编号不可修改。

### 6.3 删除资产

1. 在资产列表中选中要删除的资产
2. 点击工具栏"🗑️ 删除"按钮
3. 确认删除

## 八、折旧管理

### 8.1 折旧查询

1. 点击工具栏"📊 折旧查询"按钮
2. 选择查询日期（默认当前日期）
3. 查看资产折旧信息：
  * 资产基本信息
  * 月折旧额
  * 累计折旧
  * 净值
  * 折旧差额（首月显示）
  * 本月折旧额

### 8.2 月度折旧表

1. 点击工具栏"📅 月度折旧表"按钮
  
2. 选择年份和月份
  
3. 查看月度折旧明细：
  
  * 资产编号、名称、类别
  * 原值、正常折旧额、折旧差额
  * 当月总折旧额
  * 累计折旧、净值
  * 状态
4. 点击"导出 Excel"可导出月度折旧报表
  

### 8.3 折旧计算规则

**计算公式**：

    月折旧额 = 原值 ÷ 折旧月数
    总折旧 = 月折旧额 × 总月份
    折旧差额 = 原值 - 总折旧
    首月折旧额 = 月折旧额 + 折旧差额
    后续月份折旧额 = 月折旧额

**示例**（原值 5000 元，72 个月）：

* 月折旧额：69.44 元
* 总折旧：69.44 × 72 = 4999.68 元
* 折旧差额：5000 - 4999.68 = 0.32 元
* 首月折旧额：69.44 + 0.32 = 69.76 元
* 后续每月：69.44 元
* 最终总折旧：5000 元（净值 = 0）

**不计提折旧的情况**：

* 已提足折旧的资产
* 状态为"待报废"的资产
* 状态为"待维修"的资产

## 十、常见问题

### Q1：启动时提示缺少 openpyxl

**解决**：安装 openpyxl

    pip install openpyxl

### Q2：导入 Excel 提示"未找到有效数据"

**解决**：

1. 检查数据是否从第 2 行开始填写
2. 检查资产类别是否在预设列表中
3. 检查日期格式是否为 YYYY-MM-DD

### Q3：资产编号没有自动生成

**解决**：新增资产时，资产编号字段留空，系统会自动生成。

### Q4：折旧差额显示为 0

**说明**：V18.0 已修复此问题。折旧差额 = 原值 - (四舍五入后月折旧 × 总月份)，确保总折旧 = 原值。

### Q5：如何备份数据

**方法**：复制 `assets.db` 文件到其他位置即可。
